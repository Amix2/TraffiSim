# Tarffi sim

3 typy budynków
- dom
- fabryka
- sklep 

W domu mieszkaniec śpi, je (zużywa zasoby) 
Fabryka produkuje zasoby z obecnych w niej mieszkańców
Sklep służy jako sieć dystrybucyjna - mieszkańcy mogą zdobyć w nim zasoby 

Pętla gry:
Jak efektywnie zbudować te 3 typy budynków, żeby wszystko grało. Istotne jest mikro na drogach 

Dom zużywa zasoby za każdym razem jak wróci do niego mieszkaniec. Może jechać do sklepu żeby je usupełnić.
Sklep dostaje dostawy zasobów od fabryki
